# **App Name**: Trackit Finance

## Core Features:

- Voice Input (Bengali): Enable users to input daily expenses using voice commands in Bengali.
- Expense Categorization (AI): Automatically sort and categorize expenses into different sections (e.g., food, transportation, rent) using AI. Use an LLM tool to decide the category based on reasoning, and when necessary use information about typical expenses.
- Loan Tracking: Separate sections for tracking given loans and taken loans.
- Monthly Expense Analysis: Compare monthly expenses with monthly income, providing insights on financial health. It will suggest possible next steps for the user based on insights found using AI.
- Total Expense Display: Display the total expenses for the month.
- Monthly Reset: Automatically reset expense tracking at the start of each month while recording historical data.
- Currency Support: Ensure the application supports BDT currency.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5), suggesting trust and security.
- Background color: Very light grey (#F0F2F5) creates a clean and professional feel.
- Accent color: Teal (#009688) to highlight key actions and data points, giving the sense of precision.
- Body and headline font: 'PT Sans' (sans-serif) for a clean, readable interface. It is suitable for both headlines and body text
- Use simple, easily recognizable icons to represent expense categories.
- Design a clear, intuitive layout for easy expense input and data visualization.
- Subtle animations on data updates and category transitions to enhance user engagement.
